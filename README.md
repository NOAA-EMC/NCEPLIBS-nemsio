# NEMSIO

Performs I/O for the NCEP models using NEMS.

Code manager: Hang Lei


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang
with MPI


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-nemsio
make -j2
make install
```
