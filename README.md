# NEMSIO

Performs I/O for the NCEP models using NEMS.


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang | PGI
with MPI


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-nemsio
make -j2
make install
```
